This section documents all rei commands and their options.

## Dashboard Commands

```bash
# Show daily dashboard
rei today
rei today --context work

# Find neglected intentions
rei neglected                    # 10+ days without activity (default)
rei neglected --days 30          # 30+ days without activity
rei neglected --context work     # Filter by context

# Show weekly review
rei review week
rei review week --context personal

# Show system info
rei info

# Show version
rei version
```

The `today` dashboard includes:
- Active intentions (grouped by context when unfiltered)
- **Completed Today** section showing intentions completed today
- Today's actions
- Active blockers
- Habits due today
- Pending reflections
- Overdue and upcoming tasks

## FZF Integration

Many commands support interactive selection via [fzf](https://github.com/junegunn/fzf). When an ID argument is optional and fzf is installed, you'll get an interactive picker.

```bash
# These open fzf if ID is not provided
rei intention show
rei habit show
rei blocker resolve
```

## Output Formats

Most list commands support format options:

```bash
# Default human-readable output
rei intention list

# FZF-compatible (tab-separated)
rei intention list --format=fzf

# JSON output
rei intention list --format=json
```
