The agent commands provide AI-assisted coaching based on your data.

## Subcommands

### coach

Get general coaching.

```bash
rei agent coach [options]
```

**Options:**
- `--today` â€” Focus on today's context

**Examples:**

```bash
# General coaching
rei agent coach

# Focused on today
rei agent coach --today
```

### nudge

Get a quick suggestion for what to do next.

```bash
rei agent nudge
```

Returns a brief, actionable suggestion based on your current state.

### review

Deep review of a specific intention.

```bash
rei agent review [intention-id]
```

Provides detailed analysis and suggestions for a single intention.

### reflect

AI-assisted reflection prompts.

```bash
rei agent reflect today
rei agent reflect week
```

The agent reviews your recent activity and asks questions to help you reflect more deeply.

## Configuration

### LLM Backend

Set the backend via environment variable:

```bash
# Use Claude CLI (default)
export REI_LLM_BACKEND=claude-cli

# Use Anthropic API directly
export REI_LLM_BACKEND=anthropic-api
export REI_ANTHROPIC_API_KEY=sk-ant-...
```

### Model Override

Override the default model:

```bash
export REI_LLM_MODEL=claude-3-opus-20240229
```

## Privacy

The agent:
- Reads your local data (intentions, actions, reflections)
- Sends relevant context to the LLM
- Never modifies your data
- Processes locally when using claude-cli
