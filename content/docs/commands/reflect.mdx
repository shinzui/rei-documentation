Schedule and complete reflections. Reflections are structured review sessions.

## Subcommands

### today

Start or continue today's daily reflection.

```bash
rei reflect today [options]
```

**Options:**
- `-p, --prompt <text>` — Custom prompt text

**Behavior:**
1. Creates a reflection file at `reflections/<date>.md`
2. Opens your `$EDITOR`
3. On save, records the reflection as completed

**Examples:**

```bash
# Start daily reflection
rei reflect today

# With a custom prompt
rei reflect today --prompt "Focus on what blocked you today"
```

### week

Start or continue this week's reflection.

```bash
rei reflect week [options]
```

**Options:**
- `-p, --prompt <text>` — Custom prompt text

**Example:**

```bash
rei reflect week
```

### intention

Start a reflection focused on a specific intention.

```bash
rei reflect intention <intention-id>
```

**Arguments:**
- `<intention-id>` — Intention ID (required)

**Example:**

```bash
rei reflect intention intention_01h455vb4pex
```

### pending

List all pending (scheduled but not completed) reflections.

```bash
rei reflect pending
```

### history

Show reflection history.

```bash
rei reflect history [options]
```

**Options:**
- `-n, --limit <n>` — Number of reflections to show (default: 10)
- `-c, --context <context>` — Filter by context

**Examples:**

```bash
# Show last 20 reflections
rei reflect history -n 20

# Show reflections filtered by context
rei reflect history --context work

# Combine filters
rei reflect history --context personal -n 5
```

**Notes:**
- Intention-anchored reflections inherit context from their intention
- Daily and weekly reflections can also be filtered if they have associated context

### open

Open an existing reflection in your editor.

```bash
rei reflect open [reflection-id]
```

**Arguments:**
- `[reflection-id]` — Reflection ID (optional if fzf is available)

> **FZF Support:** If the ID is not provided and [fzf](https://github.com/junegunn/fzf) is installed, an interactive picker will let you select from recent reflections. Use `ctrl-f` to toggle between pending-only and all reflections (including completed and skipped).

**Behavior:**
- By default, shows only pending/in-progress reflections
- Press `ctrl-f` in FZF to toggle and view all reflections
- If there are no pending reflections, automatically shows all reflections

**Example:**

```bash
rei reflect open reflection_01h455vb4pex
```

### print

Print reflection content to stdout (useful for piping to other commands).

```bash
rei reflect print [reflection-id]
```

**Arguments:**
- `[reflection-id]` — Reflection ID (optional if fzf is available)

> **FZF Support:** If the ID is not provided and [fzf](https://github.com/junegunn/fzf) is installed, an interactive picker will let you select from recent reflections.

**Examples:**

```bash
# Print a specific reflection
rei reflect print reflection_01h455vb4pex

# Pipe to less for paging
rei reflect print | less

# Use fzf to select reflection
rei reflect print
```

### complete

Mark a reflection as completed.

```bash
rei reflect complete [reflection-id]
```

**Arguments:**
- `[reflection-id]` — Reflection ID (optional if fzf is available)

> **FZF Support:** If the ID is not provided and [fzf](https://github.com/junegunn/fzf) is installed, an interactive picker will let you select from in-progress reflections.

This is useful when you've edited a reflection file directly and want to mark it as done.

### cycle

Start a reflection for a completed cycle.

```bash
rei reflect cycle [cycle-id] [options]
```

**Arguments:**
- `[cycle-id]` — Cycle ID to reflect on (optional, defaults to last completed cycle)

**Options:**
- `-p, --prompt <text>` — Custom prompt text
- `--stdin` — Read reflection content from stdin
- `--actor <name>` — Creator identifier (e.g., "user", "claude-code")

> **FZF Support:** If the cycle ID is not provided and [fzf](https://github.com/junegunn/fzf) is installed, an interactive picker will let you select from completed cycles.

**Behavior:**
1. Loads context from the completed cycle (focus selections, duration, etc.)
2. Creates a reflection file with cycle summary
3. Opens your `$EDITOR`
4. On save, records the reflection as completed

**Examples:**

```bash
# Reflect on last completed cycle
rei reflect cycle

# Reflect on a specific cycle
rei reflect cycle cycle_01abc123...

# With a custom prompt
rei reflect cycle --prompt "What did you learn this cycle?"
```

### schedule

Schedule recurring or one-off reflections. Recurring reflections automatically create a new instance when the previous one is completed or skipped.

#### schedule daily

Schedule a daily recurring reflection.

```bash
rei reflect schedule daily [options]
```

**Options:**
- `-p, --prompt <text>` — Custom prompt/focus for the reflection
- `-c, --context <context>` — Context for filtering (e.g., `work`, `personal`)
- `--actor <actor>` — Actor performing this action

**Examples:**

```bash
rei reflect schedule daily --prompt "What did I accomplish today?"
rei reflect schedule daily --context work --prompt "Work review"
```

#### schedule weekly

Schedule a weekly recurring reflection on a specific day.

```bash
rei reflect schedule weekly [options]
```

**Options:**
- `-d, --day <day>` — Day of the week (default: sunday). Accepts: monday, tuesday, ..., sunday (or mon, tue, ..., sun)
- `-p, --prompt <text>` — Custom prompt/focus
- `-c, --context <context>` — Context for filtering
- `--actor <actor>` — Actor performing this action

**Examples:**

```bash
# Weekly reflection on Sundays (default)
rei reflect schedule weekly

# Weekly reflection on Fridays
rei reflect schedule weekly --day friday --prompt "Week in review"
```

#### schedule quarterly

Schedule a quarterly recurring reflection.

```bash
rei reflect schedule quarterly [options]
```

**Options:**
- `-p, --prompt <text>` — Custom prompt/focus
- `-c, --context <context>` — Context for filtering
- `--actor <actor>` — Actor performing this action

**Example:**

```bash
rei reflect schedule quarterly --prompt "Quarter review and planning"
```

#### schedule yearly

Schedule a yearly recurring reflection on a specific date.

```bash
rei reflect schedule yearly --month M --day D [options]
```

**Options:**
- `-m, --month <N>` — Month (1-12, required)
- `-d, --day <N>` — Day of month (1-31, required)
- `-p, --prompt <text>` — Custom prompt/focus
- `-c, --context <context>` — Context for filtering
- `--actor <actor>` — Actor performing this action

**Example:**

```bash
rei reflect schedule yearly --month 12 --day 31 --prompt "Year in review"
```

#### schedule on

Schedule a one-off reflection for a specific date.

```bash
rei reflect schedule on DATE [options]
```

**Arguments:**
- `DATE` — Target date in YYYY-MM-DD format (required)

**Options:**
- `-k, --kind <kind>` — Reflection kind (e.g., `daily-review`, `weekly-review`, `custom:label`)
- `-p, --prompt <text>` — Custom prompt/focus
- `-c, --context <context>` — Context for filtering
- `--actor <actor>` — Actor performing this action

**Example:**

```bash
rei reflect schedule on 2026-03-01 --kind "quarterly-review" --prompt "Q1 review"
```

#### schedule pause

Pause a recurring reflection. Paused reflections stop generating new instances.

```bash
rei reflect schedule pause [reflection-id]
```

**Arguments:**
- `[reflection-id]` — Reflection ID (optional if fzf is available)

> **FZF Support:** If the ID is not provided, an interactive picker will let you select from recurring reflections.

**Example:**

```bash
rei reflect schedule pause reflection_01abc...
```

#### schedule resume

Resume a paused recurring reflection.

```bash
rei reflect schedule resume [reflection-id]
```

**Arguments:**
- `[reflection-id]` — Reflection ID (optional if fzf is available)

> **FZF Support:** If the ID is not provided, an interactive picker will let you select from paused reflections.

**Example:**

```bash
rei reflect schedule resume reflection_01abc...
```

#### schedule cancel

Permanently cancel a recurring reflection.

```bash
rei reflect schedule cancel [reflection-id]
```

**Arguments:**
- `[reflection-id]` — Reflection ID (optional if fzf is available)

> **FZF Support:** If the ID is not provided, an interactive picker will let you select from recurring reflections.

**Example:**

```bash
rei reflect schedule cancel reflection_01abc...
```

**Notes:**
- Cancelled reflections cannot be resumed; create a new schedule instead
